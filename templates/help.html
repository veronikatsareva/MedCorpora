Поиск по лемме

Введите слово без кавычек, чтобы найти употребления всех словоформ данной лексемы.

  • врачи

    → все предложения, в которых есть словоформы врач, врачу, врачи и др.

  • врач

    → тот же набор предложений.

Поиск по словоформе

Введите слово в двойных кавычках, чтобы найти предложения с заданной словоформой.

  • "спасли"

    → все предложения, в которых употреблена словоформа спасли.

  • "красную"

    → все предложения, в которых употреблена словоформа красную.

NB: в двойные кавычки должна быть взята каждая словоформа в запросе.


  • "мама" "пришла"
  • "мама пришла"

Поиск по части речи

Введите часть речи в формате Universal Dependencies, чтобы найти все предложения с заданной частью речи.

  • NOUN

    → все предложения, в которых употреблено(-ы) существительное(-ые).

  • NUM

    → все предложения, в которых употреблено(-ы) числительное(-ые).

Комбинированный поиск

Введите лемма+POS-тег, чтобы найти все употребления словоформ с ограничением по части речи.

  • и+CCONJ

    → все предложения, в которых употреблен сочинительный союз и.

  • и+PART

    → все предложения, в которых употреблена частица и.

Сложные запросы

Корпус также поддерживает поиск N-грамм любой длины, то есть в поисковом запросе можно написать через пробел несколько токенов, заданных одним из 4 способов выше.

  • врачи нашли NOUN
  • NOUN и+CCONJ пациенты
